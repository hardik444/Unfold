## 📖 About

Unfold is a full-stack platform that helps users understand internet events, creators, brands and controversies through searchable timelines and categorized stories.

It also includes a custom CMS that allows administrators to manage content.

---

## ✨ Features

### Public Website

- Homepage
- Search Engine
- Featured Stories
- Trending Today
- Dynamic Story Pages
- Category Pages
- Responsive Design
- Dark Theme

### Admin CMS

- Secure Login

## 🚀 Run Locally

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Create `.env.local`

```env
MONGODB_URI=
JWT_SECRET=
ADMIN_EMAIL=
ADMIN_PASSWORD=
ADMIN_ACCESS_KEY=
```
