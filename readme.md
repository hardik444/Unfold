# Unfold

A full-stack platform to explore internet events, creators, brands, and controversies through searchable timelines and categorized stories. Built with Next.js, TypeScript, MongoDB Atlas, Tailwind CSS, and a custom admin CMS.

## Features

### Public Website

- Homepage
- Featured Story
- Trending Today
- Search Engine
- Category Filters
- Dynamic Story Pages

### Admin CMS

- Admin Login
- Dashboard
- Add Story
- Edit Story
- Delete Story
- Timeline Editor

### Tech Stack

Frontend:
- Next.js
- React
- TypeScript
- Tailwind CSS

Backend:
- Next.js API Routes
- MongoDB Atlas
- Mongoose

## Run locally

```bash
npm install
npm run dev
```

Create a `.env.local`:

```env
MONGODB_URI=
JWT_SECRET=
ADMIN_EMAIL=
ADMIN_PASSWORD=
```
